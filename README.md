# ZZ
This is a quorum && truffle/solidity demo project

## Usage

### Step1

see also: https://github.com/jpmorganchase/quorum-examples

```
git clone https://github.com/jpmorganchase/quorum-examples

cd quorum-examples
vagrant up


vagrant ssh

cd quorum-examples/7nodes/

init.sh
start.sh

```

### Step2

```
truffle compile
truffle migrate
truffle console
```
