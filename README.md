# ZZ
This is a quore && truffle demo project

## Usage

see https://github.com/jpmorganchase/quorum-examples

```
git clone https://github.com/jpmorganchase/quorum-examples

cd quorum-examples
vagrant up


vagrant ssh

cd quorum-examples/7nodes/

init.sh
start.sh

```

```
truffle compile
truffle migrate
truffle console
```
